# httpqd

http抢答301


使用方法：
第一步下载运行软件：

wget https://raw.githubusercontent.com/anwang520520/httpqd/master/setup

第二步开权限：

chmod 777 /root/setup


第二步运行下列代码

./setup

等安装完按键盘的Ctl+c结束进程，

再运行下列代码

nohup ./setup

访问地址：http://ip:9090
管理账号：admin
管理密码：admin
